FreeASPupload
-------------
Date: May.29/2014

- http://www.freeaspupload.net/ The good news about this script is that it is written in pure VBScript
  but you must be careful about it because it needs ADO.
  Tested myself in Windows 2003 Server with IIS 6
  

Date: Aug.20/2014
- It thought to be good as it required no server side component installation but it proved to be useless for uploads of more than
  200MB. No buffer can be configured properly at least in 32-bit servers.
